# V1.1.0 - Architecture Refactor

Release date: 2026-05-16

## Highlights

- Split oversized backend modules into focused route, task and data helper modules.
- Added daily startup scripts: `start-backend.bat` and `start-frontend.bat`.
- Added pytest coverage and GitHub Actions CI.
- Added `step07_build_db.py` to the default pipeline orchestration and Admin pipeline visibility.
- Split Admin Vue dashboard logic and extracted large view/component styles.
- Hardened React Cesium build asset relocation on Windows.
- Rewrote README and added a complete V1.1 architecture document.

## Compatibility

- Existing V1.0.0 data directory conventions remain valid.
- Existing numbered BAT files remain available for setup, pipeline, build and production-style startup.
- Daily development can now use only the two new startup scripts.

## Verification

Before release, run:

```powershell
.venv\Scripts\python.exe -m pytest
.venv\Scripts\python.exe platform\scripts\run_pipeline.py --list
.venv\Scripts\python.exe platform\scripts\run_pipeline.py --dry-run --skip 01 --skip 05
npm.cmd run type-check --prefix platform\webgis-react
npm.cmd run build --prefix platform\webgis-react
npm.cmd run typecheck --prefix platform\admin-vue
npm.cmd run build --prefix platform\admin-vue
```
