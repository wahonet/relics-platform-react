# V1.1.0 - 架构调整版

发布日期：2026-05-16

## 重点变化

- 根目录 BAT 清理为两个日常入口：`start-backend.bat` 和 `start-frontend.bat`。
- 删除旧的根目录编号 BAT，数据管线改由 `platform/scripts/run_pipeline.py` 或 Admin 管线工作台手动触发。
- 拆分 FastAPI 大入口：瓦片、DEM、Admin 任务、Admin 文物管理分别进入独立模块。
- 拆分 `data_loader.py` 中的序列化、后台查询、后台统计、普查覆盖率逻辑。
- Admin Vue 完成第一轮大文件整理：Dashboard composable 化，Relics 与 RelicEditDialog 样式外置。
- 新增 pytest、CI、开发依赖与管线 dry-run 验证。
- README、架构文档、发布说明和执行日志重新归类。

## 兼容性

- V1.0.0 的 `data/input/` 与 `data/output/` 数据约定继续保留。
- `config.example.yaml` 仍然是配置模板，首次运行时手动复制为 `config.yaml`。
- 后端 API 路径保持兼容，前端 dev server 继续通过 Vite 代理访问后端。
